Usando o github e o gitflow sem mistérios

Eu, particularmente, tive muita dificuldade para entender como usar e quais as vantagens do git para meus projetos.
Esta dificuldade, em parte, vem pela necessidade que tenho de visualizar o que estou aprendendo e, temos pouco disso neste mundo de computação, obviamente porque a linguagem é uma forma de poder e como forma de poder é dificultada.
Por causa disso, resolvi criar esta página para auxiliar àqueles que, como eu, não utilizam de forma constante estas ferramentas mas, gostariam de se beneficiar delas.
Assim, vou usar os conhecimentos que vou adquirindo com HTML e CSS para aprimorar a página e, vou fazer um passo a passo para uso do github e gitflow. Bom proveito!

Principais diferenças

Antes de iniciarmos, precisamos entender para que serve cada uma das ferramentas que vamos descrever. Todas estas ferramentas (git, github e gitflow) são aplicações para proteger o seu código, por meio de cópias em lugares distintos e por meio de controle de alterações.
